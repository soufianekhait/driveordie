# Drive or Die

## Requirements
* CLion IDE
* Cmake 3.19 x86/x64
* SDL stable version 2.0.12 (Development Libraries) 
### On Windows :
* MinGW 32/64-bit (in order to support GCC compiler on Windows systems)
* GDB for debugging
### On MacOS :
* Command Line Tools for XCode (in order to support GCC compiler on MacOS)
